# Password Strength Checker

## Description
A simple Python project that checks whether a password is Weak, Medium, or Strong based on its length.

## Features
- Takes password input from the user
- Checks password strength
- Displays Weak, Medium, or Strong

## Technology Used
- Python

## How to Run
1. Open the file in VS Code.
2. Run password_checker.py
3. Enter a password.
4. View the result.
